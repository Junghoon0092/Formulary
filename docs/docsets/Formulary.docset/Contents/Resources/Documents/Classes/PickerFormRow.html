PickerFormRow

public class PickerFormRow : FormRow, FormularyComponent

PickerFormRow is a type of FormRow that allows for picking one item from an arbitrary list of items via a UIPicker control

To use a picker form row, it is only necessary to create one, add it to a FormSection, and add that FormSection to a Form.
